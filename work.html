Meet App

An app that allows you to search by location to see what events are happening in that city. This will display a list of events pulled from the Google Calendar API with the start time, location and a button to toggle a description of the events. A graphical overlay is included to see a breakdown of type and number of events in each location.

The App is a PWA built using React, and so works offline and can be opened from desktop or smartphone screen. Sign in to the app requires a gmail account and is authenticated by OAuth2.0. and Google API Services.

Built using: React and JavaScript
