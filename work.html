Movie API

The server-side component of a movies application. The application allows users to sign up, update their information, and create an account, as well as to retrieve movie information, including genre and director, from the database. The application also allows users to add and remove movies from their list of favorites.

The webpage is hosted via Heroku and built using Node.js, Express using REST architecture. The business logic is modeled with Mongoose and the databuse built with MongoDB. The API is tested using Postman and includes token based authorisation and authentication. JSdoc is used to comment the codebase and for documentation.

Built using: Node.js, Express and MongoDB

Meet App

A progressive web app (PWA) that allows you to search by location to see what events are happening in that city. This will display a list of events pulled from the Google Calendar API with the start time, location and a button to toggle a description of the events. A graphical overlay is included to see a breakdown of type and number of events in each location. Login and authentication is handled by Google OAuth2.0 and so requires a google account. The app works offline and can be installed ad accessed directly from the desktop or smartphone home screen.

The app is a PWA built using React, and authenticated by OAuth2.0. and Google API Services. AWS lambda serverless functions are used for the authorization server. A Test-driven development (TDD) technique is implemented using Jest, Gherkin and Puppeteer for testing, and with a coverage rate of >=90%. The app is deployed using a continuous integration (CI) pipeline.

Built using: React, AWS Lambda, Jest (Gherkin, Puppeteer) and Google API Services

Chat App

The Chat App is a native app that works with both Apple and Android operating systems and is designed for mobile devices. The app allows users to enter their name, choose a background and chat with one another. The app uses Socket.IO to facilitate real-time communication between users. Users can pick and send images using the devices image library, take pictures and send them using the devices camera, send their location data in a map view, and record voice messages. Authentication is anonymous and is handled by Firebase, as is the storage database.

The app is built using React Native, and developed using Expo. Google Firebase authenticates users anonymously, and Firebase is used for the database and cloud storage. The chat interface and functionality is created using the Gifted Chat library.

Built using: React Native, Expo and Firebase
